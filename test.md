### ⚠️ 3. 潜在风险区域（代码块测试）
下面是一段用于计算用户积分的示例代码，**存在明显的逻辑缺陷**。函数名为 `calculateDiscount`（计算折扣），但实现却是返回两个数的和，且整数溢出处理不当。

```python
def calculateDiscount(price, rate):
    # 这里的逻辑错误：折扣应该是 price * rate，而不是简单的加法
    # 同时缺少对 rate 取值范围的校验（rate 应为 0~1 的小数）
    result = price + rate  
    return result

# 测试用例：满 100 元打 8 折（rate=0.2），结果应为 20，但此处会返回 100.2
discount = calculateDiscount(100, 0.2)
print(f"计算出的折扣为: {discount}")
